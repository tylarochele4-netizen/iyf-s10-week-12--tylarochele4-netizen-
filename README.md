# Deployment & Auth

This repository contains the full-stack implementation of the Week 12 curriculum, focusing on **Authentication Context**, **CORS Configuration**, and **Production-ready Deployment**.

## 🏗 Project Structure

The project is organized into a monorepo structure to separate the concerns of the server and the user interface:

- **/backend**: Node.js & Express API with MongoDB integration.
- **/frontend**: React application built with Vite and Tailwind CSS.
